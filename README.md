# Trabalho de Implementação de Algoritmos em Grafos
Neste trabalho é implementado um sistema modular para análise de propriedades de um grafo em Python. Além disso, o programa conta com um visualizador apresentando a figura do grafo resultante.

# Funções implementadas
## Verificar
  1. Conexo
  2. Bipartido
  3. Euleriano
  4. Possui ciclo
## Listar
  5. Componentes conexas
  6. Componentes fortemente conexas
  7. Um caminho Euleriano 
  8. Um caminho Hamiltoniano 
  9. Vértices de articulação
  10. Arestas ponte
## Gerar
  11. Árvore de profundidade
  12. Árvore de largura
  13. Árvore geradora mínima
  14. Ordem topológica 
  15. Valor do caminho mínimo entre dois vértces 
  16. Valor do fluxo máximo 
  17. Fecho transitvo
